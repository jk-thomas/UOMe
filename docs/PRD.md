# PRD - Shared Expense Tracker (v1)

## Goal
A mobile-friendly web app for a fixed group to log purchases and compute balances automatically.

## Rules
- Single group (v1)
- Fixed members (no auth)
- Every purchase is split evenly across all members
- All purchases are public within the group
- Edit/delete purchases only to correct mistakes

## MVP
- Add purchase (payer, amount, optional description)
- List purchases
- Balance + settlement suggestions
- Edit/delete purchase
